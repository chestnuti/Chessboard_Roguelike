# 构成 / 酸性棋盘式肉鸽战术游戏
## 开发任务清单

版本：0.1
依据： [GameDesignDoc_技术需求与实现文档.md](GameDesignDoc_技术需求与实现文档.md)

说明：
- 以下任务按推荐开发顺序排列，前序任务完成后再进入后续任务
- 优先级采用 `P0` / `P1` / `P2`
- `P0` 表示阻塞核心玩法或原型验证，`P1` 表示对完整体验关键，`P2` 表示增强或优化项

## 阶段划分建议

### 原型阶段

目标：先跑通最小战斗闭环。

建议完成任务：1 至 10

### 垂直切片阶段

目标：验证完整一局体验。

建议完成任务：11 至 15

### 优化与扩展阶段

目标：提升可玩性、稳定性与内容量。

建议完成任务：16 至 18

---

## 关键里程碑

### 里程碑 1：最小可玩原型

- 玩家可以在棋盘地图中移动
- 进入敌人格可触发近战
- 双属性值可正常变化
- 敌人可行动并产生压制/免疫结果
- 能量可掉落、持有并用于 3×3 地块转换

### 里程碑 2：完整单局闭环

- PCG 可稳定生成可达地图
- 多房间推进正常
- 远程敌人按四向直线规则行动
- 友伤与跨阵营清除链成立
- HUD 与提示完整可读

### 里程碑 3：可测试版本

- 核心参数可调
- 关键数值可记录
- 可根据测试反馈快速迭代
- 具备基础音画表现

## 备注

- 若资源紧张，优先保留 `P0` 任务，`P1` 任务中可先砍 UI 美术和音效表现，`P2` 任务可在核心玩法稳定后再补。
- 地块转换能量相关任务必须在 PCG 深度调优前完成，否则后续平衡难以收敛。
